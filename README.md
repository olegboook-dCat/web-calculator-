# Калькулятор фасадных материалов

Онлайн-калькулятор для расчёта материалов и стоимости монтажа фасада:
сайдинг, фиброцемент, композитные панели или металлопрофиль. Работает
прямо в браузере — не нужно ничего устанавливать, регистрироваться или
куда-то загружать данные.

**Открыть калькулятор:** https://olegboook-dcat.github.io/web-calculator-/

## 🎮 Бонус: мини-игра «Стройка»

В репозитории есть мини-игра `house-game.html` — пошаговый симулятор
строительства дома. Нужно пройти все этапы от изысканий до отделки,
учитывая реальные нюансы: тип грунта и уровень воды, выбор фундамента,
набор прочности бетона (28 суток), армопояс для газобетона, усадку сруба,
обсаду окон, вентиляцию, сезонность работ и бюджет. Каждый участок
генерируется случайно, в конце — оценка дома и разбор ошибок.

**Играть:** https://olegboook-dcat.github.io/web-calculator-/house-game.html
(или просто откройте файл `house-game.html` в браузере).

## Что умеет

- Готовые пресеты под тип фасада (сайдинг, фиброцемент, композит,
  металлопрофиль) с типовыми размерами панели, шагом профиля и % отходов —
  можно менять под свой проект.
- Ввод стен и проёмов (окон/дверей) — площадь проёмов автоматически
  вычитается из площади стен.
- Расчёт количества панелей, погонных метров профиля (с учётом углов
  здания) и крепежа — с учётом заданного % отходов.
- Расчёт стоимости по каждой позиции и итоговая смета.
- Проверка введённых данных: некорректные значения (например, нулевая
  ширина стены) подсвечиваются, а под результатами появляется понятное
  предупреждение.
- Сохранение и загрузка нескольких именованных проектов — хранятся
  локально в браузере (localStorage), никуда не отправляются.
- Автосохранение черновика: если случайно закрыть вкладку или обновить
  страницу, несохранённая работа не потеряется.
- Отмена/повтор последних действий (кнопки «Отменить» / «Повторить»
  или `Ctrl+Z` / `Ctrl+Shift+Z`).
- Экспорт сметы в CSV (открывается в Excel с русскими буквами).
- Печать или сохранение в PDF — отдельный аккуратно оформленный отчёт.

## Как пользоваться

1. Откройте ссылку выше (или файл `index.html` прямо в браузере — интернет
   для расчётов не нужен).
2. В шаге 1 укажите название объекта, клиента и выберите тип фасада —
   калькулятор подставит типовые параметры материала. При необходимости
   поправьте размеры панели, шаг профиля, % отходов и количество углов
   здания вручную.
3. В шаге 2 добавьте стены (ширина, высота, количество одинаковых стен).
4. В шаге 3 добавьте окна и двери — их площадь будет вычтена из площади
   стен.
5. В шаге 4 укажите цены на панель, профиль и крепёж — результат и смета
   обновятся автоматически.
6. Нажмите «Сохранить», чтобы добавить проект в список сохранённых (это
   отдельно от автосохранения черновика — так можно хранить несколько
   объектов и возвращаться к ним).
7. «Экспорт CSV» — скачает смету в виде таблицы для Excel.
8. «Печать / PDF» — откроет диалог печати браузера с аккуратным отчётом;
   там же можно сохранить смету как PDF-файл.

## Важно про данные

Все данные (сохранённые проекты, черновик, введённые цены) хранятся
только в вашем браузере, на вашем устройстве — они не отправляются ни на
какой сервер. Если очистить данные браузера или открыть калькулятор в
другом браузере/на другом устройстве, сохранённые проекты будут
недоступны.

## Технические детали

Один файл `index.html` — HTML, CSS и JavaScript без сборки и внешних
зависимостей. Чтобы обновить сайт после правок, достаточно закоммитить и
запушить изменения в файле `index.html` — GitHub Pages опубликует их
автоматически в течение минуты-двух.
